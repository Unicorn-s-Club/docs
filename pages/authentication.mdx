<Tip>
  **Token da API**: Sua credencial de acesso à API da Unicornify. Esta chave
  identifica sua conta e autoriza suas requisições.
</Tip>

  <Step title="Copie seu Token">
    <Card>

## Como usar seu token

Após obter seu token, você pode começar a integrar com nossa API. Lembre-se de incluir o token em todas as requisições em Query Parameters usando `api_token`:

Adicione o token como um parâmetro de query string na requisição requisições:

```bash
?api_token=seu_token'
```

## Exemplo de requisição

```bash
curl --location 'https://api.unicornify.com.br/api/public/v1/products?api_token=seu_token' \
--header 'Accept: application/json'
```
